# TSM Hub

**Live:** [hub.truesourcemetals.com](https://hub.truesourcemetals.com)

Free, open metals data platform by [TrueSource Metals](https://truesourcemetals.com). Official prices, reserves, production statistics, and market news — sourced directly from LME, LBMA, SHFE, and USGS.

## Features

- **Official Prices** — LME (USD/t), LBMA (USD/oz), SHFE (RMB/t), updated daily via automated pipeline
- **Reserves & Production** — USGS Mineral Commodity Summaries 2026 data for every metal
- **Market Digest** — Curated industry news filtered by metal and topic
- **Glossary** — 80+ terms covering metals trading, blockchain, and RWA tokenisation
- **Metal Pages** — Dedicated page for each commodity with prices, charts, producers, and data

## Data Sources

All data sourced from primary, official sources — no estimates, no editorial bias.

- **LME** — London Metal Exchange official settlement prices
- **LBMA** — London Bullion Market Association gold and silver prices
- **SHFE** — Shanghai Futures Exchange settlement prices
- **USGS** — Mineral Commodity Summaries 2026 (reserves, production, trade)

## Automation

Prices and news update automatically via GitHub Actions. No manual intervention required.
